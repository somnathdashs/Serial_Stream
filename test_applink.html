  3. The app should open and show a verification success message

Alternative Testing Methods

For Android:

Run these ADB commands:

adb shell am start -W -a android.intent.action.VIEW -c android.intent.category.BROWSABLE -d "serialstream://verify=true" com.sddev.serial_stream

adb shell am start -W -a android.intent.action.VIEW -c android.intent.category.BROWSABLE -d "https://serial.stream/verify/?v=true" com.sddev.serial_stream

adb shell am start -W -a android.intent.action.VIEW -c android.intent.category.BROWSABLE -d "@https://serial.stream/verify/?v=true" com.sddev.serial_stream

For iOS:

Run these commands with the iOS Simulator:

/usr/bin/xcrun simctl openurl booted "serialstream://verify=true"

/usr/bin/xcrun simctl openurl booted "https://serial.stream/verify/?v=true"

/usr/bin/xcrun simctl openurl booted "@https://serial.stream/verify/?v=true"
